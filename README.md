## 주제   
번역기 어플리케이션을 만든다.  
## 목표  
- 번역과 관련된 아이디어로 딥러닝 모델을 만든다.
- 아이디어를 구체화하여 계획을 수립한다.
- 데이터를 직접 구축 하거나 task에 적절한 오픈 데이터를 찾는다.
- 실현 가능한 아이디어로 실제 모델을 만들고 라이브 데모까지 완성한다.
- Pytorch를 이용해 아이디어와 task에 맞게 모델을 처음부터 설계한다.
